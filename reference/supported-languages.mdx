## Choosing the Right Model

<Note>
  For the broadest language support, use **Whisper Large V3 Turbo** (99 languages) or **Qwen3 ASR** (30+ languages including Chinese dialects).
</Note>

<Info>
  **Apple Speech** requires no model download but depends on which dictation languages you've installed in System Settings. Check System Settings → Keyboard → Dictation to see available languages.
</Info>

### Recommended Models by Use Case

| Use Case | Recommended Model | Why |
|----------|------------------|-----|
| English transcription | Parakeet TDT 0.6B | Fastest with top accuracy |
| Multilingual (many languages) | Whisper Large V3 Turbo | 99 language support |
| Chinese + Asian languages | Qwen3 ASR | Includes Chinese dialects |
| No model download | Apple Speech | Built-in, uses system languages |
| Resource-constrained | Whisper Tiny | Smallest download (~150 MB) |

## Adding Languages to Apple Speech

If you're using the Apple Speech model, follow these steps to add language support:

1. Open **System Settings**
2. Navigate to **Keyboard** → **Dictation**
3. Click the **+** button to add a new dictation language
4. Select your desired languages
5. Petal will automatically detect and use these languages for transcription

<Warning>
  Apple Speech is only available on macOS 26 or later. If you're on an earlier version of macOS, this model will not appear in Petal.
</Warning>

## Model Download Sizes

All models except Apple Speech require downloading before first use:

- **Apple Speech**: No download required (built-in)
- **Qwen3 ASR**: ~2.5 GB
- **Parakeet TDT 0.6B**: Not specified (CoreML format)
- **Whisper Large V3 Turbo**: ~1.6 GB
- **Whisper Tiny**: ~150 MB
- **Voxtral Mini 3B (BF16)**: ~9.4 GB
- **Voxtral Mini 3B (8-bit)**: ~4.6 GB

<Info>
  Models are downloaded from Hugging Face on first use and cached locally for future transcriptions.
</Info>
